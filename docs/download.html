What is CBZ format?

CBZ is a standard comic book archive format (basically a ZIP file with images). It's supported by all major comic reader applications and preserves the original image quality.

Benefits:

  • Read offline on any device
  • Full-resolution images
  • Works with your favorite comic reader app
  • Includes metadata (title, page count, etc.)
